# API仕様の確認方法

## Write API Server: GraphQLのSDLを確認する方法

```shell
# CLIで確認する場合
$ makers view-command-sdl
# ファイルに出力する場合
$ makers export-command-sdl ./schema.sdl
```

## Read API Server: GraphQLのSDLを確認する方法

```shell
# CLIで確認する場合
$ makers view-query-sdl
# ファイルに出力する場合
$ makers export-query-sdl ./schema.sdl
```
